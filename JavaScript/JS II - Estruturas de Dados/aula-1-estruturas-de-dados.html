Estruturas de Dados

Estruturas de dados servem para criar objetos mais complexos e tratar os dados de forma mais eficiênte

Exemplos mais Comuns

Lista Ligada

Coleção ordenada de valores onde cada elemento aponta para o próximo

Array

Coleção de elementos que podem ser identificados por um índice

Queue

Coleção de elementos onde o primeiro elemento a ser retirado deve ser o primeiro a ser adicionado

also called FIFO (first-in/first-out)

Funciona como uma fila

Stack

Coleção de elementos onde o primeiro elemento a ser retirado deve ser o último a ser adicionado

Also called LIFO (last-in/first-out)

Funciona exatamente como uma pilha de folhas de papel

Tree
